GitHub and Git Commands

Terminal

In this section, you will learn about GitHub and the various Git Commands that you will be using in Terminal. GitHub is an open source control software that allows many people to collaborate on the same/seperate web pages at once.

  • git init : creates a git repository in the current directory
  • git add : adds the file to the commit - UNKNOWN
  • git commit : modifications within the tree - UNKNOWN
  • git checkout : change branch in repository
  • git push : upload local repository to remote repository - ASK CLAIFICATION
  • git status : prints the state current working branch as well as the staging area
